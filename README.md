Text Processor

A simple Python based text processing application that cleans text, extracts email addresses, analyzes text statistics and compresses processed text into a binary file using zlib.


Concepts used:

clean_text(text)
Removes unnecessary punctuation and extra spaces.
extract_emails(text)
Uses regex to find email addresses in the text.
count_words(text)
Counts total words.
count_unique_words(text)
Counts unique words using Python sets.
count_sentences(text)
Counts sentences using punctuation delimiters.
compress_text(text)
Compresses text using zlib and saves it as compressed_text.bin.
